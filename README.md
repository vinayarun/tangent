# Tangent

**AI that helps you think, not just answer.**

Tangent is an AI-assisted research workspace that augments traditional chat with structured cognitive views built on a robust conversation engine and knowledge graph.

---

## 🚀 Overview

Generative AI enables powerful conversations, but most interfaces remain linear. Human thinking is non-linear — involving exploration, branching, synthesis, and revision.

Tangent bridges this gap by transforming AI conversations into structured knowledge systems while preserving the simplicity of chat.

---

## ✨ Key Features

- 🧭 Tangent Graph — Visual exploration of conversation branches
- 🔬 Research Compass — Structured investigation workflow
- ⚖️ Reasoning Graph — Claims, evidence, counterarguments mapping
- ⏳ Time-Concept Navigator — Dual axis conversation navigation
- 🧠 Insight Compression — Multi-level knowledge summarization
- 🌐 Model-agnostic AI integration
- 💾 Local-first knowledge storage

---

## 🧠 Philosophy

Tangent is built on principles from:

- External Cognition Theory
- Dual Process Thinking (System 1 / System 2)
- Knowledge Graph Research
- Intelligence Analysis Workflows
